# GPU-DS-ML-Poster

This project explored the role of GPUs in modern data science by investigating how machine learning computational demands scale with dataset size and how GPU hardware characteristics influence performance. This was a group project undertaken as part of a module within MSc Data Science and Business Analytics.

The project involved:

* Generating and analysing synthetic datasets of increasing size.
* Measuring and comparing the computational cost of three machine learning models:

  * Decision Tree
  * Random Forest
  * K-Nearest Neighbours (KNN)
* Performing repeated timing experiments to evaluate how model training and prediction times scale as dataset size increases.
* Analysing GPU benchmark data (2010 - 2022) to examine trends in computational performance over time.
* Investigating the relationship between GPU benchmark performance and:

  * Market value
  * Thermal Design Power (TDP)
* Creating data visualisations to explore performance, cost, and energy-efficiency trade-offs.
* Interpreting the implications of hardware selection for the scalability, accessibility, and sustainability of data science workloads.

The project combined exploratory data analysis, machine learning performance evaluation, benchmark score analysis, and data visualisation to assess the impact of GPU hardware on modern data science applications.
